# Changelog

## v0.2.0 — 2026-05-22

First public binary release. Phase 21 M1.

- `sov` CLI (TypeScript runtime, Bun-compiled standalone)
- `sov-tui` (Go, Bubble Tea, cross-compiled per platform)
- Default agent bundle (skills + agents + prompts)
- Platforms: `darwin-arm64`, `darwin-x64`, `linux-x64`
- One-line installer at `install.sh`
- `sov upgrade` auto-detects binary mode and re-runs the installer
